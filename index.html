Tags and their summary

The summary tag(<summary>)

The summary tag works hand-in-hand with the details tag. It acts as the label or title for the hidden content. Think of it as a sign on the magic box, telling users what's hidden inside. This tag is crucial because it's the clickable element that users interact with to expand or collapse the details section. It's always visible, even when the details are hidden.

The details tag(<details>)

The details tag in HTML is like a magic box for your web page. It creates a collapsible section of content that stays hidden by default. Users can't see what's inside until they click on a trigger. This tag is perfect for hiding lengthy information, specifications, or anything that might clutter the main view..

The paragraph tag(<p>)

This tag defines a block of text that is typically formatted as a paragraph. It creates white space before and after the content, making it visually distinct from other elements.

The img tag(<img>)

This tag inserts an image into your web page. The src attribute specifies the path to the image file. You can also add alternative text using the alt attribute to describe the image content for accessibility purposes.

The anchor tag(<a>)

This tag creates a hyperlink that, when clicked, directs the user to another web page, section of the current page, or another resource. It defines the link's destination (href attribute) and the displayed text (content between the tags).

10 elements with opening and closing tags

  1. <span>: Groups text for styling without meaning change (like highlighting).
  2. <strong>: Bold text for emphasis (often keywords).
  3. <ul>: Unordered list with bullet points.
  4. <li>: An item within an ordered or unordered list.
  5. <main>
  6. <article>: Self-contained content piece (like a blog post).
  7. <aside>
  8. <section>
  9. <div>
  10. <button>

Ways to add CSS to a webpage

There are 3 ways to add CSS to a webpage:

  • Inline Styles: Add style attribute directly to the HTML tag (limited use).
  • Internal Stylesheet: Create a <style> tag within the <head> of your HTML.
  • External Stylesheet: Create a separate .css file and link it to your HTML with a <link> tag in the <head>.
